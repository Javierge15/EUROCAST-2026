# WebRTC Station

Este proyecto utiliza Docker para ejecutar una aplicación WebRTC.

## Instrucciones para lanzar la aplicación

1. **En el vehículo:** Ejecuta el siguiente comando para iniciar los contenedores:
   ```bash
   docker-compose up -d
   ```

2. **En la estación:** Busca la dirección IP del contenedor en tu navegador para acceder a la aplicación WebRTC.
